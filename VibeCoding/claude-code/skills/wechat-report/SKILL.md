# 微信报告发送（Server酱）

当用户请求将报告发送到微信时，使用此技能。通过 Server酱 推送到微信公众号消息。

## 触发条件

当用户的请求中包含以下关键词时触发：
- "发微信"
- "微信发给我"
- "发到微信"
- "微信通知"
- "通知微信"

## 工作流程

### 第一步：生成报告内容

根据当前会话完成的任务，生成 Markdown 格式的报告：

```markdown
## 任务摘要
[简要描述完成了什么任务]

## 完成的工作
- [工作项 1]
- [工作项 2]

## 注意事项
[需要用户注意的问题]
```

### 第二步：调用发送脚本

```bash
~/.claude/scripts/send-to-wechat.sh "报告内容" "标题"
```

或通过 stdin：

```bash
cat << 'EOF' | ~/.claude/scripts/send-to-wechat.sh
## 任务摘要
完成了 xxx 功能...
EOF
```

### 第三步：确认结果

- `✅ 报告已发送到微信` - 成功
- `❌ 发送失败: ...` - 失败

## Server酱 Markdown 支持

支持标准 Markdown 语法：
- **粗体**、*斜体*
- 标题 (# ## ###)
- 列表（有序和无序）
- 代码块
- 链接
- 图片

## 内容限制

- 标题最长 256 字符
- 内容最大 32KB
- 脚本会自动截断超长内容

## 配置

SendKey 已配置在脚本中：
```
~/.claude/scripts/send-to-wechat.sh
```

如需修改 SendKey，编辑脚本中的 `SENDKEY` 变量。

## 示例

### 示例 1：任务完成后发送

用户："帮我检查服务器状态，完成后发微信给我"

### 示例 2：发送自定义内容

用户："把今天的工作总结发到微信"
